Dans le cadre de son nouveau projet SENDIGITOUR, Simplon Sénégal vous demande de lui réaliser un landing page ergonomique avec wordpress.

Le site doit avoir une identité visuelle cohérente dans son ensemble, y compris le logo, les couleurs du thème, les polices avec la charte graphique de Simplon Sénégal.

La page doit contenir les informations suivantes :

  • un slider avec minimum trois images
  • le nom et le slogan de simplon
  • ajouter les informations du projet
  • ajouter les liens pour les réseaux sociaux

Pour des besoins de communication, vous devez ajouter une deuxieme page pour la publication des articles du projet.
